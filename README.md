# Chronicle Frontend

A premium‑styled frontend for the Chronicle investigative journalism platform.

## Overview
Chronicle is an open‑source publishing solution built for long‑form, data‑driven stories. This frontend provides a modern, glass‑morphism UI with smooth animations, dark/light mode support, and a responsive layout powered by Vite + React.

## Features
- Vibrant design with glass‑morphism cards and micro‑animations
- Zig‑zag entrance animations for page sections
- Dark/Light mode toggle
- Interactive landing page with hero, feature highlights and animated footer
- Full editor, reader, document viewer, collaboration workspace
- Fast performance – stories load under 2 seconds even with large content
- Responsive layout across mobile, tablet and desktop

## Getting Started

```bash
git clone https://github.com/Hafsa2122/Chronicle-Publish-Investigative-Journalism.git
cd Chronicle-Frontend
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Building for Production

```bash
npm run build
npm run preview
```
